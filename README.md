# Blood-Bank
# 🩸 Blood Bank Management System

> A web-based system designed to manage blood donors, blood availability, and requests efficiently.

---

## 🚀 About the Project

The Blood Bank Management System is a dynamic web application built using PHP and MySQL that helps streamline the process of blood donation and distribution. It provides a centralized platform where donor details can be stored, blood availability can be tracked, and users can request blood when needed.

The goal of this project is to reduce manual effort and improve accessibility in managing critical blood bank operations.

---

## ⚡ What it does

* Stores donor information
* Tracks available blood groups
* Handles blood requests
* Displays donor and blood details
* Simplifies management for administrators

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS
* **Backend:** PHP
* **Database:** MySQL
* **Server:** XAMPP / WAMP

---

## 🧠 How it works

The system connects a user-friendly interface with a backend database. When a donor registers or a request is made, the data is stored in MySQL and processed using PHP. The application retrieves and displays relevant information such as available blood groups and donor details in real time.

### 🔧 Requirements

* XAMPP / WAMP server
* PHP installed
* MySQL database

### ⚙️ Steps

1. Copy the project folder to `htdocs`
2. Start Apache and MySQL from XAMPP
3. Import the database file into phpMyAdmin
4. Open browser and go to:

   ```
   http://localhost/blood-bank-system/
   ```
